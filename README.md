# Сайт для управления личными заметками

Добавляй и изменяй заметки для личного пользования.
Есть возможность использовать любое кол-во аккаунтов

## Фишки

- авторизация
- заметки
- категории
- БД

### Стек

- Django 4.2
- Postgres
- Redis (кэш заметок и категорий)

### Доп. библиотеки
- pytils (slugify)
- python-dotenv
